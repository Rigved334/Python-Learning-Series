Basic Indexing
Example (String: "Python"):
string[0] returns 'P'.
string[4] returns 'o'.
string[-4] returns 't'.

Slicing
Examples
string[0:3] extracts "Pyt" (indices 0, 1, 2).
string[3:6] extracts "hon" (indices 3, 4, 5).

Using the Step Value
Example:
string[0:6:2] results in 'Pto'

Reversing a String
Example: string[::-1] outputs "nohtyP".
